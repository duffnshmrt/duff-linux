## Installers

To run Calamares:

```
su

calamares
```


To run the tui installer just open a terminal and type:

```
sudo d77void-installer
```

Note: To maintain the configuration of the live iso, during install, choose local instead of network install.

During install, add your user to the storage group. That way udiskie will automount disks.

## 1st run:

After install, run the script d77-welcome:

```
d77-welcome
```

That way the d77void repo will be added to /etc/xbps.d and you can add easily other things like steam, flatpak, etc.

# Keybinds

super -> drop down wayfire shell menu

super + return -> terminal

super + e -> see all desktops

super + ctrl + cursor -> move from desktop (up, left, down, right)

super + ctrl + shift + cursor -> move active window from desktop (up, left, down, right)

super + q -> close window

super + shift + q -> logout

super + a -> menu

super + f -> file manager

super + m -> mail

super + t -> lock screen

super + w -> web browser
