Hello
Thank you for trying d77void.

To run the installer just open a terminal and type:

```
sudo d77void-installer
```

Note: 
To maintain the configuration of the live iso, during install, choose local instead of network install.

During install, add your user to the storage group. That way udiskie will automount disks.

## Keybinds

alt + shift + return -> terminal

alt + shift + q -> logout

alt + shift + c -> close window

alt + t -> tile mode

alt + f -> floating mode

alt + m -> focus mode

alt + p -> dmenu

alt + r -> gmrun

alt + s -> scratchpad

alt + x -> powermenu

alt + w -> brave

super + b -> swap wallpaper

super + c -> control pannel

super + d -> rofi

super + f -> pcmanfm

super + l -> slock

super + m -> geary

super + p -> scrot

## Hacking

To add/remove autostart programs edit the file ~/.dwm/autostart.sh
