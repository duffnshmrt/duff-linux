Hello
Thank you for trying d77void.

To run the tui installer just open a terminal and type:

```
sudo d77void-installer
```

Note: 
To maintain the configuration of the live iso, during install, choose local instead of network install.

During install, add your user to the storage group. That way udiskie will automount disks.

# Calamares

To use the dead simple Calamares installer, just press <alt + d> and type calamares.


## general Keybinds

alt + shift + return -> terminal

alt + shift + q -> logout

alt + shift + c -> close window

alt + shift + r -> reload configuration


## d77 Keybinds

alt + a -> scratchpad

alt + b -> swap wallpaper

alt + d -> rofi

alt + e -> mousepad

alt + m -> mail

alt + t -> lock screen

alt + w -> brave

alt + x -> powermenu

alt + z -> yakuake alike scratchpad

super + f -> pcmanfm

super + j -> jgmenu

super + p -> scrot

super + r -> gmrun
