# Hyprland

3 approaches for the same super WM;

1) Hyprland vanilla, with some tweaks like Fuzzel as app list and power menu and Waybar as OS status bar.

2) Noctalia; a professional shell with excellent aesthetic and funcionality.

3) DankMaterial; a professional shell with excellent aesthetic and funcionality.

All of them have their keybinds; Hyprland and Noctalia share most of it, but DankMaterial has his own tweaks and keybinds, so read the README file inside the ISO.


## Install

To install acquire root

```
su

calamares

```

or 

```
su

d77-installer

```

and follow instructions on your screen!


Have fun!
